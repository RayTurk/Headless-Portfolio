DEPLOYMENT INSTRUCTIONS
================================================================================

1. Copy theme to WordPress installation:
   cp -r wordpress/theme/ /path/to/wordpress/wp-content/themes/portfolio-headless/

2. Copy plugins to WordPress installation:
   cp -r wordpress/plugins/* /path/to/wordpress/wp-content/plugins/

3. Activate in WordPress Admin:
   - Activate theme from Appearance > Themes
   - Activate plugins from Plugins menu

4. Configure in WordPress Admin:
   - Settings > Headless Settings
   - Set Frontend URL
   - Set Revalidation Secret
   - Enable ISR Webhooks (if using)

5. Test GraphQL endpoint:
   - Visit /graphql on your WordPress installation
   - Run test queries to verify functionality
